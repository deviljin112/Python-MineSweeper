# Python Minesweeper

Simple Python minesweeper for cmd.
:snake: Written on Python 3.9.0

## Requirements

:snake: Python 3.7+

Pretty Table Module
`pip install prettytable`

Colorama Module
`pip install colorama`

## Usage

- Clone the repo
- Extract all the files
- Open CMD
- CD into your extracted folder
- `python main.py`
- Enjoy
